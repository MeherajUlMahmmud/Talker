# Talker

Yet Another Chat App

## Installation

-   Go into `server` folder

```bash
cd SERVER
```

-   Install dependencies

```bash
npm install
```

-   Add `ENV_VARIABLES`

```bash
MONGO_URI=
LOCAL_MONGO_URI=mongodb://localhost:27017/talker
PORT=8000
JWT_SECRET=
```

-   Run the app

```bash
npm start
```
